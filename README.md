## Description

Boilerplate for react/redux app

## Prerequisites

Yarn

## Installation

```bash

$ yarn install

```

## Running the app

```bash

$ yarn start

```

## Test

```bash
# run unit tests
$ yarn test

# run unit tests and watch for changes
$ yarn test:watch

```

## Storybook

```bash
# run storybook
$ yarn storybook

# build storybook
$ yarn build-storybook

```

## Build

```bash
# build app
$ yarn build


```
