## Description

Boilerplate for react/redux app

## Prerequisites

Yarn

## Installation

```bash

$ yarn install

```

## Running the app

```bash

$ yarn start

```

## Test

```bash
# run unit tests
$ yarn test

# run unit tests and watch for changes
$ yarn test:watch

```

## Build

```bash
# build app
$ yarn build


```
